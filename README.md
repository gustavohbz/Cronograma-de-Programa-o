# Cronograma de Programação
Um cronograma para aprendizagem de linguagens de programação,em que o usuário escolhe as liguagens e cria-se um cronograma para a semana separando por horários,a quantidade de aulas e fazendo uma distribuição dessas aulas de maneira igual para serem vistas 


Etapas:
- Etapa 1: Esqueleto(concluído)
Criar o esqueleto(HTML,CSS e JS)

- Etapa 2 : Cálculos Essenciais(12/03/26 a 15/03/26 )
criar as funções que farão os cálculos e armazenamento de dados do input

- Etapa 3 : Estilização CSS(15/03/26 a 20/03/26)

- Etapa 4 : Implementação de Features(22/03/26 - 22/04/26)
implementação de features como:
Adição de cursos com carga horaria;
tempo ate concluir curso com  data precisa se seguido o cronograma corretamente;
Cálculo de Horas totais estudadas de cada matéria;
Janelas de Alerta para:
Muitas horas disponíveis no dia(+5h)
Usuário coloca a opção de curso no cronograma e substitui as materias,ou adiciona.
Dicas: Hora de praticar no cronograma junto com as aulas,exemplo
se um bloco de 1h,mas ele tem 3 blocos ao dia,tirar 25% so para Codewars ou Prática de Debugging
